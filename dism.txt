Restore To .ISO Level Files: Windows 11

0.) Download The Windows 11 .ISO x64 Installer From Microsoft.
1.) Create C:\WINNING And Rename dism.exe to idism.exe in the Installer Files So We're Using A Clean Dism 
Executable Just In Case That What We Have Is Borked/Corrupted.
2.) idism.exe /mount-wim /wimfile:install.wim /mountdir:C:\WINNING /index:1 (Extracts The Files To A Folder)
3.) idism.exe /image:C:\WINNING /cleanup-image /restorehealth (<~ oye)

4.) Below Is The Different Versions Of The Installer In The .ISO File. (So Choose The Version That You Have)

/index:number

C:\Win11_24H2_English_x64\sources>idism.exe /get-imageinfo /imagefile:install.wim
