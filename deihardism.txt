0.) Download The Windows 11 .ISO x64 Installer From Microsoft.
1.) Create C:\WIMMING And Rename dism.exe to idism.exe in the Installer (\sources) Files So We're Using A Clean Dism 
Executable Just In Case That What We Have Is Borked/Corrupted.
2.) idism.exe /mount-wim /wimfile:install.wim /mountdir:C:\WIMMING /index:1
3.) idism.exe /image:C:\WIMMING /cleanup-image /restorehealth
3.1) if C:\WIMMING needs to be remounted : idism.exe /remount-image /mountdir:C:\WIMMING
4.) idism.exe /commit-image /mountdir:C:\WIMMING or idism.exe /commit-image /mountdir:C:\WIMMING /CheckIntegrity
6.) idism.exe /online /cleanup-image /restorehealth /source:C:\WIMMING\Windows /limitaccess

Below Are The Different Versions Of The Installer In The .ISO File. (So Choose The Version That You Have)
